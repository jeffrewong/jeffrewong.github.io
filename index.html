State Redistricting System

The goal of this group project was to create a system that can rewrite voting district lines such that the new districts are more fair, compact, and representative of the demographics that live in the districts. The system utilizes AngularJS, HTML, and CSS for the front-end. The back-end uses Java to create a RESTful API for processing client actions such as redistricting a particular state map and saving the results. Saved maps are serialized and deserialized using JDBC and stored in a MySQL database.

  • Date Completed: May 2019
  • Created for: CSE 308 Software Engineering
  • Link to project gitlab here (Project repository is currently set to private by request of instructor. Please contact me for more details about the project.)

Twitter Clone

The goal of this group project was to replicate the basic functionality of Twitter (user accounts, posts, liking/sharing, media upload, and search) on a cloud enviornment and attempt to handle the amount of traffic that Twitter had in its early stages. An emphasis was placed on the creation of the RESTful API instead of the frontend, so a barebones HTML and minimal CSS page was used as the user interface. The back-end uses ExpressJS and various modules along with with MongoDB to handle requests and user posts, accounts, and media storage. The clone achieved around 600 concurrent requests handled per second with a 95 percent tail latency of around 160ms.

  • Date Completed: Fall 2020
  • Created for: CSE 356 Cloud Computing
  • Link to project gitlab here

SBML

This course project was a supplementary project to the what the course was teaching about how programming languages go from characters written on a document to executed instructions. The project aim was to create a clone of SML (called Stony Brook ML) plus/minus some features using Python as the runtime enviornment. The project required the use of the Ply library for parsing of user programs and also required the creation of nodes of a Abstract Syntax Tree to simplify the semantic checks and execution of the program. The language supports for/while loops, if/else statements, function definitions, variable definitions, blocks, and print statements.

  • Date Completed: Spring 2020
  • Created for: CSE 307 Principles of Programming Languages
